# Design System — lokalnewww.pl

## 1. Założenia marki

**lokalnewww.pl** to marka tworząca nowoczesne strony internetowe dla lokalnych firm.

Brand powinien być:

- minimalistyczny,
- nowoczesny,
- profesjonalny,
- prosty w odbiorze,
- technologiczny, ale przyjazny dla małych firm.

Główne skojarzenia: **lokalność, widoczność w Google, szybka realizacja, profesjonalna obecność online**.

---

## 2. Logo

### Wersja podstawowa

Logo składa się z:

- ikony **pina mapy w stylu 3D**,
- logotypu tekstowego: `lokalnewww.pl`.

### Kolorystyka logotypu

- `lokalne` — ciemny granat,
- `www` — intensywny niebieski,
- `.pl` — neutralny szary.

### Zastosowanie

Logo powinno być używane głównie w:

- headerze strony,
- stopce,
- materiałach social media,
- wizytówkach,
- faviconie jako sam pin.

### Zasady

Nie należy:

- rozciągać logo,
- zmieniać proporcji,
- dodawać nowych efektów,
- zmieniać kolorów poza wariantami brandowymi,
- używać logo na zbyt chaotycznym tle.

---

## 3. Kolory

### Primary

```css
--blue: #1F6BFF;
--blue-dark: #0A3FD9;
--blue-light: #6EA8FF;
```

### Neutralne

```css
--text-dark: #071426;
--text-gray: #6F7680;
--background-light: #F5F7FA;
--white: #FFFFFF;
```

### Zastosowanie kolorów

- `#1F6BFF` — przyciski CTA, linki, akcenty, ikony.
- `#0A3FD9` — hover dla przycisków, ciemniejszy gradient pina.
- `#6EA8FF` — jasne akcenty, highlighty, gradienty.
- `#071426` — główne nagłówki i tekst.
- `#6F7680` — tekst pomocniczy, `.pl`, opisy.
- `#F5F7FA` — jasne tła sekcji.

---

## 4. Typografia

### Font główny

Rekomendowany font: **Inter**

Fallback:

```css
font-family: Inter, Arial, sans-serif;
```

### Nagłówki

```css
font-weight: 800;
letter-spacing: -0.04em;
line-height: 1.05;
color: #071426;
```

### Tekst podstawowy

```css
font-weight: 400;
line-height: 1.7;
color: #6F7680;
```

### Styl komunikacji

Komunikacja powinna być:

- prosta,
- konkretna,
- zrozumiała dla lokalnych przedsiębiorców,
- bez przesadnego technicznego języka.

Przykład:

> Strony internetowe, które pomagają zdobywać klientów z Google.

---

## 5. Przyciski

### Primary Button

Używany dla głównych akcji, np. „Bezpłatna wycena”.

```css
.btn-primary {
  background: #1F6BFF;
  color: #FFFFFF;
  border-radius: 12px;
  padding: 14px 24px;
  font-weight: 700;
  box-shadow: 0 12px 30px rgba(31, 107, 255, 0.28);
}

.btn-primary:hover {
  background: #0A3FD9;
  transform: translateY(-2px);
}
```

### Secondary Button

Używany dla akcji drugorzędnych, np. „Zobacz realizacje”.

```css
.btn-secondary {
  background: #FFFFFF;
  color: #1F6BFF;
  border: 1px solid rgba(31, 107, 255, 0.3);
  border-radius: 12px;
  padding: 14px 24px;
  font-weight: 700;
}

.btn-secondary:hover {
  border-color: #1F6BFF;
  box-shadow: 0 10px 25px rgba(7, 20, 38, 0.08);
}
```

## 6. Karty UI

Karty powinny mieć miękki, nowoczesny wygląd.

```css
.card {
  background: #FFFFFF;
  border: 1px solid #F1F5F9;
  border-radius: 24px;
  padding: 24px;
  box-shadow: 0 12px 35px rgba(7, 20, 38, 0.04);
}
```

### Zastosowanie kart

- korzyści,
- usługi,
- realizacje,
- pakiety cenowe,
- opinie klientów.

---

## 7. Layout strony

### Ogólne zasady

- dużo białej przestrzeni,
- maksymalna szerokość contentu: `1200–1280px`,
- sekcje oddzielone jasnym tłem lub subtelną linią,
- duże nagłówki,
- wyraźne CTA.

### Kontener

```css
.container {
  max-width: 1280px;
  margin: 0 auto;
  padding: 0 20px;
}
```

### Sekcje

```css
.section {
  padding: 96px 0;
}
```

Na mobile:

```css
.section {
  padding: 64px 0;
}
```

---

## 8. Header

Header powinien być minimalistyczny i czytelny.

### Elementy

- logo po lewej,
- prosta nawigacja pośrodku lub po prawej,
- przycisk CTA po prawej.

### Styl

```css
.header {
  background: rgba(255, 255, 255, 0.85);
  backdrop-filter: blur(16px);
  border-bottom: 1px solid #F1F5F9;
  position: sticky;
  top: 0;
  z-index: 50;
}
```

---

## 9. Hero section

Hero powinno od razu komunikować ofertę.

### Rekomendowany nagłówek

> Nowoczesna strona, która pomaga zdobywać klientów.

### Podtytuł

> Projektujemy szybkie, responsywne i skuteczne strony WWW dla firm usługowych, sklepów i lokalnych biznesów.

### CTA

- Główne: `Bezpłatna wycena`
- Drugorzędne: `Zobacz realizacje`

### Element wizualny

Po prawej stronie warto użyć mockupu strony na desktopie i telefonie.

---

## 10. Ikony

Ikony powinny być:

- proste,
- liniowe,
- zaokrąglone,
- w kolorze `#1F6BFF`,
- osadzone w jasnoniebieskich boxach.

Przykład:

```css
.icon-box {
  width: 48px;
  height: 48px;
  border-radius: 16px;
  background: rgba(31, 107, 255, 0.08);
  color: #1F6BFF;
}
```

Rekomendowane ikony:

- map pin,
- rocket,
- search,
- smartphone,
- shield check,
- clock.

- dużo białej przestrzeni,
